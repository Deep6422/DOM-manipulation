# DOM Manipulation using JavaScript

This project demonstrates basic DOM manipulation using JavaScript.
It shows how to interact with HTML elements dynamically without
reloading the webpage.

## Technologies Used
- HTML
- JavaScript

## Features
- Read and update text from DOM
- Change styles using JavaScript
- Show and hide elements
- Create and remove elements dynamically
- Handle user input
- Simple counter application

## Conclusion
This project helps in understanding the basics of DOM manipulation
used in interactive web applications.
